Implementación acotada para medir impacto rápido.

  • Canal
    1 canal: Webhook/API (recomendado) o Email.
  • Clasificación
    3 categorías: Soporte / Ventas / Facturación.
  • Reglas
    Prioridad + datos críticos + excepción por “duda”.
  • Acción
    1 destino: Sheets / ticketera / CRM (según caso).

    Logs por ejecución + resumen semanal (simple).

Guardrails (anti-alucinación)

    Si faltan datos → no actúa. Pide información.
  • 2
    Si hay duda → pide confirmación. Human-in-the-loop.
  • 3
    Salida determinista: JSON validado (menos sorpresas).
